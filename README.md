# The Blog

## Installation

Clone project from this repo:

```bash
  git clone https://github.com/Dandy-CP/the-blog.git
```

Enter to project folder:

```bash
  cd the-blog
```

Install packages Project with npm:

```bash
  npm install
```

Run the development server localy on your machine:

```bash
  npm run dev
```
